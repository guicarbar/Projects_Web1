Sobre

Este é um site troll. A ideia dele é dificultar tudo para o usuário, como uma charada para descobrir o final do site. No entanto, para o criador, vulgo eu, ele foi desenvolvido para testar minhas habilidades em JavaScript, já que todas as funções serão feitas em JS. Além disso, minha intenção é fazer tudo com JS neste site, até mesmo de maneira desnecessária, já que algumas das minhas ideias poderiam facilmente ser realizadas com HTML e CSS. Mas quero forçar minhas habilidades, então aproveitem bastante meu site troll.
